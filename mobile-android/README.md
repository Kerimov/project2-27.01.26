# Медицинский ассистент — Android (Google Play)

Expo-приложение для **Android** и публикации в Google Play. Полный паритет с iOS-версией в `mobile-ios/`.

| Папка | Назначение |
|-------|------------|
| `mobile-android/` | Android, Google Play |
| `mobile-ios/` | iOS, App Store |

## Быстрый старт

```bash
cd mobile-android
npm install
npm run start:lan   # Metro, порт 8081
```

Backend в корне: `npm run dev` (порт 3000).

## Скрипты

| Команда | Описание |
|---------|----------|
| `npm run android` | Expo + Android |
| `npm run start:lan` | LAN для физического устройства |
